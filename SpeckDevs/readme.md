# SpeckDevs - IT Service & Solution

Welcome to **SpeckDevs**! This is the official repository for the SpeckDevs website, showcasing our IT services and solutions. Our platform offers a wide range of services, including web development, UI/UX design, digital marketing, and more, all aimed at empowering businesses to thrive in the digital age.

---

## 🌟 Features

- **Dynamic Navigation**: Smooth and responsive navigation for a seamless user experience.
- **Services Overview**: Comprehensive details about the IT services offered.
- **Portfolio Showcase**: Highlighting our past projects with an engaging gallery.
- **Interactive Pricing Section**: Explore pricing options for various services.
- **Contact Form**: Easy-to-use form for inquiries and quotes.
- **Responsive Design**: Fully optimized for all devices (desktop, tablet, mobile).

---

## 📁 Project Structure

```
project/
│
├── index.html           # Main HTML file

├── header.js            # Header interaction scripts
├── price.js             # Pricing section interactivity
├── select.js            # Custom dropdown logic
├── form.js              # Contact form logic
├── images/              # Images used across the website
```

## 🎨 Customization

### Fonts and Icons
- **FontAwesome** is used for icons. Update the `<link>` tag in the `head` section for newer versions if required.

### Styling
- Modify `style.css` for global styles.
- Update `select.css` for dropdown-specific styling.

### Scripts
- Update `header.js`, `price.js`, or `form.js` for functionality enhancements.
